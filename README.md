This project is inspired by and a slice of the incredible "Traveling Landing Page" concept by designer **Arif Hossain**. I aimed to replicate the essence of the original design while adding my own improvisations and personal touches during the implementation process.

## ## Key Features

This project includes several beautifully animated key sections:

* ‚úÖ **Hero Section**: A captivating main banner with a video background and interactive stats.
* ‚úÖ **Destination Highlights**: A gallery to showcase featured travel locations in Norway.
* ‚úÖ **How It Works**: A step-by-step explanation of the platform's process with illustrations.
* ‚úÖ **Flexible Travel Plans**: A card-based view for personalized travel packages.
* ‚úÖ **Testimonials**: A user testimonials section with an infinite scroll effect to build trust.
* ‚úÖ **Smooth Scrolling**: Implemented using **Lenis** for a fluid and seamless navigation experience.
* ‚úÖ **Responsive Design**: A fully optimized layout for both desktop and mobile devices.

## ## Tech Stack

* **Framework**: [Next.js](https://nextjs.org/) (React)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Animation**: [GSAP (GreenSock Animation Platform)](https://gsap.com/)
* **Smooth Scroll**: [Lenis](https://lenis.studiofreight.com/)
* **Deployment**: [Vercel](https://vercel.com/)

## ## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/spherule-demo.git](https://github.com/your-username/spherule-demo.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd spherule-demo
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

5.  Open [http://localhost:3000](http://localhost:3000) in your browser.
